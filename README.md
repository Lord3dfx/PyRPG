INSTALLING:

1. Clone repository:
```bash
git clone https://github.com/username/project-name.git
cd project-name
```
2.Install venv
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# or
venv\Scripts\activate     # Windows
```
3. Install dependecies
``` bash
pip install -r requirements.txt
```
4. Run project
```bash
python main.py
```
